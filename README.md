# 🚀 LoopPrompt AI

An AI-powered Prompt Optimization Agent built with n8n, Groq LLM, and Conversation Memory using Loop Engineering principles.

## 📌 Overview

LoopPrompt AI helps users improve prompts through multiple refinement iterations instead of answering them directly.

The agent:
- Analyzes the prompt
- Scores prompt quality
- Identifies missing information
- Generates an optimized prompt
- Asks follow-up questions
- Uses memory to refine prompts over multiple iterations
